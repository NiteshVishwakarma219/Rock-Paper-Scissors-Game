# 🎮 Rock Paper Scissors Game

🚀 A simple interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript.  
This project helps in understanding JavaScript logic building, DOM manipulation, and basic UI interactivity.

---

## 📌 Overview
This project simulates the classic Rock Paper Scissors game where the user plays against the computer.

It is built to strengthen core frontend and JavaScript fundamentals such as:
- Conditional logic
- Event handling
- DOM manipulation
- Random number generation
- UI updates dynamically

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

---

## ✨ Features
- 🎮 Play against computer
- 🧠 Random computer choice generation
- 🖱️ Click-based interaction
- 📊 Live score tracking
- 🎨 Simple and clean UI
- ⚡ Instant result display (Win / Lose / Draw)
